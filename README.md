# Chocolate Sales Analytics Project (MySQL)
This is a sales analysis project of a startup chocolate company (Blessed Chocolate Store) using MySQL. The goal is to extract key insights from the sales data to enable the CEO and business stakeholders in long term evaluation of the business and support informed decision-making processes.

## Blessed Chocolate Store Business Objectives;
- Write queries to extract information about data
- Analyze sales performance across different time periods
- Identify top-performing salespeople and products
- Categorize sales into performance tiers (Low, Mid, High, Exemplary)
- Evaluate customer purchasing behavior
- Measure revenue trends and sales distribution

## Overview about the data
The data source is Hypothetical in nature. It results from manually desiging a database then adding tables in MySQL. Data is manually recorded into respective fields.
The dataset consists of Four tables namely;
- Sales; Holds the transactions data.
- Employed Salespersons; Holds salespersons details.
- Products;  Holds product details.
- Geography; Holds geographical details.

## Sample SQL concepts covered
- The SELECT statement usage
- Sorting and Filtering
- Aggregations
- Date functions
- Ordering and ranking
- Categorization
- Joining of data tables

## Deliverables
- Retrieval of data based on certain conditions
- Updated the Database, renamed tables and columns
- Grouped data into categories i.e from Low to high.
- Identified best performing salespeople based sales made.
